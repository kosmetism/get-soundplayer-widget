SoundPlayer.js

Embeddable SoundCloud players built with deku and deku-sounplayer as component layer and soundcloud-audio as HTML5 Audio API wrapper.
This is a proof of concept in building component-based, high-quality embeddable widgets with deku.js. Check the accompanying article for more details.
